Overview

The Simple Chat App is a basic chat application that allows users to send and receive text messages in real-time. It is designed to be a minimalistic and easy-to-use chat platform for casual conversations.

Features

  • Real-Time Messaging: Users can send and receive messages in real-time without the need to refresh the page.
  • Message Separation: Chat messages are displayed with a horizontal line to visually separate them.
  • Auto-Sync: The chat history is automatically synchronized with the server every 2 seconds to display new messages without manual page refresh.
  • User-Friendly Interface: The app has a clean and user-friendly interface with an input box for typing messages and a send button.

Usage

Sending Messages:

Type your message in the input box at the bottom of the chat interface. Click the "Send" button or press "Enter" to send the message.

Receiving Messages:

  • Messages sent by other users will appear above the input box.
  • New messages are automatically displayed in real-time without refreshing the page.

Chat History:

  • The chat history is displayed in reverse order, with the most recent messages at the top.

Auto-Sync:

  • The chat history is automatically updated every 2 seconds to show new messages.

Styling

The chat app has a simple and clean style with the following visual elements:

  • White chat container with a border and shadow.
  • Light gray horizontal lines separating chat messages.
  • Input box for typing messages with rounded corners.
  • Blue "Send" button with white text.
  • Footer section with social links and copyright information.

Customization

You can customize the appearance and behavior of the app by modifying the provided HTML and CSS files. Additional features and enhancements can also be implemented to suit your specific requirements.

Deployment

To deploy the Simple Chat App, follow these steps:

  1. Set up a server environment (e.g., Node.js) to host the app files.
  2. Note: You must install expres.js if not already done. Do that with "npm install express"
  3. Place the server.js, chat.txt, index.html, and chat.js files in the server directory.
  4. Start the server using the node server.js command.
  5. Access the chat app in a web browser by navigating to the server's URL.
